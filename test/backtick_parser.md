# 백틱 파서 테스트

## 1. 단일 백틱 (기존 동작 회귀 가드)

- 인라인 코드: `console.log("hi")`
- 라벨 칩(의미색): `[사실]` `[추정]` `[의견]` `[모름]`
- 라벨 칩(자주 쓰는 말): `[중요]` `[TODO]` `[참고]` `[경고]` `[팁]`
- 라벨 칩(아무 단어): `[고양이]` `[디자인]` `[리뷰]`

## 2. 더블 백틱 — 새로 지원 (CommonMark 표준)

- 마크다운 syntax 시연: `` `[단어]` ``
- 백틱 보여주기: `` `code` ``
- JS 템플릿 리터럴: `` `Hello ${name}` ``
- 셸 백쿼트: `` `date` ``

## 3. 더블 + 단일 혼합 (이전엔 깨졌던 케이스)

백틱으로 감싼 `` `[단어]` `` 가 색칩이 된다. `[사실]` `[추정]` `[의견]` `[모름]` 같은 의미 라벨, `[중요]` `[TODO]` 같은 자주 쓰는 말은 색이 미리 정해져 있고, 그 밖에 아무 단어나 칩이 된다.

## 4. 트리플 백틱 (CommonMark 표준)

- 더블 백틱을 보여주려면: ``` `` `code` `` ```
