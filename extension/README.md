# Markdown eBook Reader for VS Code

Read your Markdown files like a premium eBook directly inside VS Code.

## Features

- **Premium Layout**: Beautiful pagination, typography, and themes (Paper, Sepia, Gray, Black).
- **Interactive Controls**: Table of Contents, settings panel, full keyboard support, and 3D turn animations.
- **Two-way Sync**: Changing check boxes inside the eBook updates the original `.md` source file, and editing the source updates the preview in real-time.
- **Local Link Support**: Clicking a link to another `.md` file opens the document directly inside VS Code.

## Usage

1. Open any Markdown (`.md`) file.
2. Right-click inside the editor and select **"Ebook Reader: Open Preview"** (or use the Command Palette `Ctrl+Shift+P`).
